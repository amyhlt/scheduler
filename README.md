## Dependencies
 - axios
 - classnames
 - react
 - react-dom
 - react-scripts
 - react-test-renderer
 -  @testing-library/react-hooks
## Setup
Install dependencies with `npm install`.
## Running Webpack Development Server
npm start
go to "http://localhost:8000/" in browser
## Running Jest Test Framework
npm test
## Running Data API server
npm start
